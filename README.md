# AI ATS Resume Analyser using n8n

## Overview
This project is an AI-powered ATS Resume Analyser built using n8n automation workflows and OpenRouter AI models.

The workflow automates:
- Resume upload
- PDF parsing
- ATS analysis
- Resume scoring
- Gmail report delivery

---

## Workflow

Resume Upload Form
→ PDF Resume Parser
→ AI ATS Analyser
→ Gmail ATS Report Sender

---

## Features

- ATS Resume Scoring
- AI-Based Resume Analysis
- Skill Gap Detection
- Automated Gmail Reports
- PDF Resume Parsing
- Workflow Automation

---

## Technologies Used

- n8n
- OpenRouter API
- Gmail API
- PDF Parsing
- AI Automation
